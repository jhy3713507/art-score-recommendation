# 山东省2026年美术类本科志愿推荐系统

基于山东省教育招生考试院官方公布的 2025 年艺术类本科批（第一次志愿）投档数据开发，旨在为 2026 年美术与设计类考生提供科学、便捷的志愿填报参考。

## 🚀 核心功能

1. **智能分数计算**：
   - 支持输入美术专业分（0-300）和文化课分数（0-750）。
   - 严格按照山东省官方公式计算综合分：`综合分 = 专业分 × 1.25 + 文化分 × 0.5`。

2. **个性化院校推荐**：
   - 自动匹配综合分低于或等于用户分数的院校专业。
   - 按分差提供安全边际建议：
     - **非常稳妥**：用户分高出投档线 > 10分
     - **较稳妥**：用户分高出投档线 5-10分
     - **有风险**：用户分高出投档线 < 5分

3. **动态参考分配置**：
   - 支持手动设定“参考专业分”（默认 234 分）。
   - 实时反推各专业的文化课最低要求，帮助考生评估文化课压力。

4. **高级筛选与搜索**：
   - **省份筛选**：系统自动智能识别院校所属省份，支持按省份快速过滤。
   - **关键词检索**：支持对院校名称、专业名称进行模糊搜索。
   - **分页展示**：推荐结果每页 10 条，支持智能分页跳转。

5. **院校详情查询**：
   - 点击推荐结果或在查询模块搜索，可查看院校详细投档信息。
   - 弹窗展示同校其他专业的投档线，支持快速点击切换查看。

## 🛠 技术架构

- **前端**：
  - **Vue.js 3 (CDN)**：轻量级响应式框架，处理数据绑定与交互逻辑。
  - **Tailwind CSS**：原子化 CSS 框架，实现高效的响应式 UI 设计。
  - **HTML5/CSS3**：标准网页结构与自定义淡入动画。
- **数据处理**：
  - **Python (Pandas)**：用于将原始 `.xls` 格式的官方投档数据清洗并转换为前端高效读取的 `.json` 格式。
- **部署**：
  - **GitHub Pages**：纯静态部署，支持全球快速访问。

## 📖 使用指南

### 本地预览
1. 确保已安装 Python 3。
2. 在项目根目录运行：`python -m http.server 8000`。
3. 浏览器访问：`http://localhost:8000`。

### 数据更新
1. 将最新的官方 Excel 数据放入 `data/` 目录。
2. 运行 `python scripts/convert_data.py`。
3. 脚本会自动更新 `data/scores.json`。
